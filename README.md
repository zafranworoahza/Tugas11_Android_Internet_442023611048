Berikut Penjelasan singkat mengenai perbedaan karakteristik antara level perlindungan izin Normal Permission (seperti internet access) dengan Dangerous Permission dari sudut pandang interaksi pengguna (runtime prompt) :

Normal Permission itu kayak kamu masuk ke sebuah toko — kamu nggak perlu izin khusus dari kasir, cukup masuk aja. Sistem Android langsung kasih izin otomatis waktu app di-install, tanpa nanya ke pengguna sama sekali. Contohnya akses internet — app bisa langsung konek ke jaringan tanpa kamu pernah lihat popup apapun.
Dangerous Permission kebalikannya — ini kayak ada orang asing yang minta buka dompet kamu. Android sengaja "memaksa" app untuk berhenti sebentar dan nanya langsung ke kamu di saat runtime (saat app sedang jalan), bukan waktu install.
